# graduate_backend
This is my FYP's backend of my undergraduate studies.<br>
This project is based on Golang1.18 and Gin.<br>
To use this project:<br>
1, cd src<br>
2, go mod tidy<br>
3, go build -o main<br>
4, ./main<br>
Then you can find it is running on the 8080 port.<br>
